One-time setup — configure your Firebase connection

🔥 Why Firebase?
QuizArena uses Firebase Realtime Database so the admin can control chapters, questions, and settings — and students see changes instantly. Without it, the app still works in offline mode with default questions.
1
Go to console.firebase.google.com → Create a project (free)
2
Enable Realtime Database → Start in test mode
3
Go to Project Settings → Your apps → Web app → Copy the config object
4
Paste the values below and click Save
⚙️ Firebase Configuration
Paste your Firebase config values here. They'll be saved in your browser and used by the app.
